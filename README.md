# 🌐 CommUnity – Local Service Discovery Platform

**CommUnity** is a full-stack web application that helps users discover and connect with trusted local service providers — from tech experts to home repair professionals. It offers an intuitive interface to browse services by category and location, read genuine user reviews, and build community trust through a transparent rating system.

---

## ✨ Key Features

### 🛍️ Service Listings
- Add and manage service listings with images
- Categorize services by type, city, and state
- View detailed service profiles

### 👥 Community Interaction
- Add and read genuine reviews for each service
- Rate services with a star-based system
- Build trust through community feedback

### 🔐 User Experience
- Secure login and signup using JWT authentication
- Easy-to-use interface for browsing and filtering services
- Responsive design for desktop and mobile use

---

## 🛠️ Tech Stack

- **Frontend:** ReactJS, Tailwind CSS  
- **Backend:** NodeJS, ExpressJS, MongoDB  
- **Authentication:** JWT, Bcrypt  
- **Image Uploads:** Cloudinary  
- **Data Handling:** TanStack Query, Axios  
- **Other:** CORS for secure cross-origin access
